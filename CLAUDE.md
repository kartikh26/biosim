## Project Overview

biosim is a pharmacokinetically-grounded human biology simulator combining:
- A learned VAE (Variational Autoencoder) World Model for simulating biological dynamics
- Model-based Reinforcement Learning for protocol optimization
- An LLM agent layer for optimizing fitness and pharmacological interventions
